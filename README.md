# django-blog 

A simple and clean blog application created using Python's Django framework.

## Features
1. Public - Anyone can create Posts
2. User Authentication - Users can register, login and logout
3. Create, Edit and Delete Posts

## Technologies
- Python 
- Django
- Sqlite3
- HTML, CSS, JavaScript
- Bootstrap
